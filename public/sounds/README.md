# Animal Sound Files

This directory should contain 12 animal sound MP3 files for the soundboard.

## Required Files

You need to add the following MP3 files to this directory:

1. `dog.mp3` - Dog bark/woof sound
2. `cat.mp3` - Cat meow sound
3. `cow.mp3` - Cow moo sound
4. `duck.mp3` - Duck quack sound
5. `sheep.mp3` - Sheep baa sound
6. `pig.mp3` - Pig oink sound
7. `rooster.mp3` - Rooster crow sound
8. `chicken.mp3` - Chicken cluck sound
9. `horse.mp3` - Horse neigh sound
10. `frog.mp3` - Frog ribbit sound
11. `lion.mp3` - Lion roar sound
12. `owl.mp3` - Owl hoot sound

## Where to Find Animal Sounds

### Free Sound Libraries

1. **Freesound.org** (https://freesound.org/)
   - Search for animal sounds
   - Free with attribution (check individual licenses)
   - High-quality community-uploaded sounds

2. **Zapsplat** (https://www.zapsplat.com/)
   - Free sound effects library
   - Requires free account
   - Extensive animal sound collection

3. **BBC Sound Effects** (https://sound-effects.bbcrewind.co.uk/)
   - Free for personal/educational use
   - Professional quality recordings

4. **SoundBible** (http://soundbible.com/)
   - Royalty-free animal sounds
   - Various licenses available

### Tips for Selecting Sounds

- **Duration:** Keep sounds between 1-3 seconds for quick playback
- **Format:** MP3 format is recommended (but WAV, M4A also work)
- **File Size:** Smaller files (<500KB) load faster
- **Quality:** 44.1kHz sample rate is ideal for web
- **Volume:** Try to normalize volume across all sounds

## After Adding Sounds

1. Make sure all 12 MP3 files are in this directory
2. Test the soundboard in your browser
3. The app will automatically preload and cache these sounds
4. Sounds work offline after first load (PWA feature)

## Verification

Run this command to check if all sound files are present:

```bash
npm run check-sounds
```
