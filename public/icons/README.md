# PWA Icons

This folder contains placeholder icons for the PWA.

## Current Icons
- `icon-192x192.png` - 192x192 placeholder (purple square)
- `icon-512x512.png` - 512x512 placeholder (purple square)
- `apple-touch-icon.png` - 180x180 for iOS (purple square)

## To Replace
Create custom icons with:
- An animal or soundboard-themed design
- Use the theme color (#4F46E5) or your preferred color scheme
- Export as PNG with transparency if desired

You can use tools like:
- Figma/Sketch for design
- [PWA Asset Generator](https://github.com/elegantapp/pwa-asset-generator)
- [RealFaviconGenerator](https://realfavicongenerator.net/)
